CinExplorer es una single page application (SPA) desarrollada como Trabajo Final Académico. El objetivo es mostrar los conceptos de diseño web moderno (HTML5, CSS3), interactividad con JavaScript, y la integración de una API pública para mostrar contenido dinámico.

Características Clave
Integración con API: Obtiene y muestra dinámicamente películas populares y estrenos utilizando la API de TheMovieDB (TMDB).

Navegación Interactiva: Los enlaces de la barra superior (Populares, Estrenos) activan la consulta de diferentes endpoints de la API sin recargar la página.

Botón de Exploración: Permite cargar la lista de películas populares en la sección del carrusel.

Formulario de Contacto con Validación: Incluye un formulario completo en la sección de Contacto, con validación de datos obligatoria en el lado del cliente (JavaScript), verificando campos vacíos y el formato de email.

Diseño Responsivo: Estructura basada en Flexbox y consultas de medios (si las has añadido) para asegurar una visualización óptima en distintos dispositivos.

Tecnologías Utilizadas:
HTML5 (Estructura Semántica).

CSS3 (Estilos y Diseño).

JavaScript (ES6+) (Interactividad y Lógica de API/Validación).

Font Awesome: Para iconos visuales (como el icono de la claqueta en el logo).

TheMovieDB (TMDB) API: Fuente de datos de películas.
